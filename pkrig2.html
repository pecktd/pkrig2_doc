pkrig2.rig_base module¶

class pkrig2.rig_base.BaseRig(meta_node, mod_name, desc, write_side)¶

Bases: object

A base class to all rigging modules in pkrig. A BaseRig object is initialized with a group as a Meta Node. All conponents those are created using init_node method are connected to the Meta Node as a component of the Rigging Module.

The concept of the Meta Node is a collector of all nodes in the rigging module.

The created Meta Node is mostly used as a Ctrl_Grp. Ex. armrig = BaseRig(‘ArmRig’, ‘A’, True) # ArmRig_L_Grp is initialized as a Meta Node.

Parameters:
  • meta_node (prc.Dag/None) – A DAG node is used as a meta node. If None, an empty group is created as a meta node.
  • mod_name (str) – A module name that is applied to the end of the component part. Use ‘’ if don’t want to assign module name.
  • desc (str) – A descriptive word to add to all nodes in the module.
  • write_side (bool) – If True side of the rigging node will be written as a rigging attribute name.

The number of detail joints is doubled from span U.

Template joints for detail joints are needed. The amount of template joints is the number of span U.
