pkrig2.utils.add_space_blender(target_nodes_and_attr_names, cons_func, constrained_node, controller, attr_name, name, desc, index, side)¶

Creates space blender using a constraint node.

Ex:
add_space_blender(
[{local_ori: ‘local’}, {head_ori: ‘head’}, {world_ori: ‘world’}], prc.orient_constraint, eye_ctrl_ori, eye_ctrl, ‘orientTo’, ‘Spec’, ‘EyeRigA’, None, prc.Side.left)

Script uses local_ori, head_ori and world_ori nodes as orient constrain targets. Adds an orientTo attribute to eye_ctrl with ‘local’, ‘head’ and ‘world’ as its enum values. Adds orient constraint node to eye_ctrl_ori node. Uses condition nodes to control constrain weights. Names the created condition nodes with the given info. The name part is generated with this format:

{name}{enum_attr}{constrain_type}{controller_type}
Parameters:
  • target_nodes_and_attr_names (list) – A list of dictionaries that have target node as keys and attribute enum names as values.
  • (prc.orient_constraint/prc.point_constraint/ (cons_func) – prc.parent_constraint):
  • constrained_node (prc.Dag) –
  • controller (prc.Dag) –
  • attr_name (str) –
  • name (str) – Is added to the beginning part of the name part.
  • desc (str) – Is added to the end part of the name part, mostly used as a module description.
  • index (None/int/str) –
  • side (prc.Side) –
Returns:

A list of named condition nodes(prc.Node).
Return type:

list

pkrig2.utils.add_udoipp(ctrl)¶

Adds Up/Down, Out/In and Pull/Push controller to the given ctrl. ctrl.ty controls Up/Down, ctrl.tx controls Out/In and ctrl.tz controls Pull/Push. Amplifiers for each channel are added to shape node of the given controller.

Uses a clamp node to extract postive/negative values from ty and tx. ty+ = up ty- = down tx+ = out tx- = in tz+ = pull tz- = push

Parameters:ctrl (prc.Dag) –
Returns:Up/Down clamp node. prc.Node: Out/In clamp node. prc.Node: Up multiplier. prc.Node: Down multiplier. prc.Node: Out multiplier. prc.Node: In multiplier.

Creates a controller with a series of parent groups. Cursor dedicates how far the controller is moved away from the root group. Uses Mult Matrix and Decompose Matrix to accumulate transformations from the controller to the root group. Attributes cursorTranslateXYZ are created to the controller to contain the outputs of the accumulation.

Parameters:
  • shape (prc.CurveParam) –
  • compo (str) – A name of the current rig component.
  • index (str/int) – An index of current component.
  • side (str/prc.Side) – A side of current component.
  • mod_name (str) – A name of module concate with its description.
Returns:

prc.Dag: An offset group. prc.Dag: A constrain group. prc.Dag: A zero group. prc.Node: A multiply matrix node. prc.Node: A decompose matrix node.
